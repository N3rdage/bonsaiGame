The classic children's rhyme has a strict initiator/response structure:

- **Initiator:** "See you later, alligator."
- **Response:** "In a while, crocodile."

If I'm saying goodbye first, use the alligator line. If Drew has already said
the alligator line, respond with the crocodile line. Don't swap them — the
rhyme's joke is in the call-and-response, and using the response as the
opener reads as off.

**Why:** Drew called this out after I closed a session with "In a while,
crocodile" when I should have used "See you later, alligator" since I was
the one signing off. Tiny detail, easy to get right once noticed.

**How to apply:** When wrapping a session and Drew has already said the
alligator line (e.g. "see you later alligator"), my reply uses the crocodile
line. When I'm initiating the goodbye, I use the alligator line and Drew
gets the crocodile rejoinder.
